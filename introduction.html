Introduction

What is aShellYou?

aShellYou is a modified version of ashell, a local ADB shell, with Shizuku integration for system API access. It improves upon ashell by supporting Material You and Material Design 3, offering a more modern and user-friendly experience.


Features

  • Modern Design

  • Fast and efficient

  • Works without root
